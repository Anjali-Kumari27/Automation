
# **Manual Test Case Report**

## **Online E-Commerce Website**

---

## **1. Introduction**

This document contains the **manual test cases** for the Online E-Commerce Website.
The goal of testing is to ensure that users can **browse products, place orders, make payments, and manage their accounts** smoothly.

All testing is performed **manually without using any automation tools**.

---

## **2. Scope of Testing**

The following modules of the e-commerce website are tested:

* User Registration
* User Login
* Product Search and Browse
* Add to Cart
* Checkout Process
* Payment Gateway
* Order History
* Logout Functionality

---
