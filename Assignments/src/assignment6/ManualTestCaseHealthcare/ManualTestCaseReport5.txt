# **Manual Test Case Report**

## **Healthcare System**

---

## **1. Introduction**

This document contains the **manual test cases** for the Healthcare System.
The purpose of testing is to ensure that the system allows **patients, doctors, and administrators** to manage healthcare services smoothly, securely, and without errors.

All testing is performed **manually without using any automation tools**.

---

## **2. Scope of Testing**

The following modules of the Healthcare System are tested:

* User Registration
* User Login
* Patient Record Management
* Appointment Booking
* Doctor Availability
* Prescription Management
* Logout Functionality
