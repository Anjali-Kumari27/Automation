# **Manual Test Case Report**

## **Travel Booking Platform**

---

## **1. Introduction**

This document contains the **manual test cases** for the Travel Booking Platform.
The main purpose of testing is to ensure that users can **search, book, and manage travel services** easily and without errors.

All test cases are executed **manually without using any automation tools**.

---

## **2. Scope of Testing**

The following features of the Travel Booking Platform are tested:

* User Registration
* User Login
* Search Travel Options
* Book Ticket
* Payment Process
* View Booking Details
* Cancel Booking
* Logout Functionality
